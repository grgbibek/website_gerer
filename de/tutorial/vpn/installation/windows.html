Beginne den Setup-Prozess

Starte die Installation durch einen Doppelklick auf die Setup.exe. Im folgenden wird der Wizard zur Installation erscheinen. Fahre mit einem Klick auf „Next“ fort.

Bitte beachten: Windows fragt in manchen Fällen nach, ob Du die VPN-Installation wirklich durchführen möchtest, ob die Anwendung auf das System zugreifen darf und ob die SpyOFF VPN-Software installiert werden soll. Bestätige diese Meldungen, um die Installation fortsetzen zu können.

Akzeptiere die Nutzungsbedingungen
33.333333333333% Complete

Setze die Installation fort. Lies bitte die Nutzungsbedingungen durch und klicke „I accept the terms in the License Agreement“ an. Dadurch stimmst Du unseren Nutzungsbedingungen zu und kannst die Installation durch einen Klick auf „Next“ fortführen.

Bitte beachten: Sofern Du den Nutzungsbedingungen nicht zustimmst und „I do not accept the terms in the License Agreement“ anklickst, kann die VPN-Installation nicht fortgesetzt werden.

Wähle den Installationsort
50% Complete

Wähle nun einen Ordner zum Speichern der für SpyOFF benötigten Daten aus. Ein Pfad ist standardmäßig hinterlegt, kann jedoch mit einem Klick auf „Browse“ geändert und manuell auf Deine Bedürfnisse angepasst werden. Sofern Du Dich für einen Installationspfad entschieden hast, bestätige bitte Deine Eingabe, indem Du auf „Next“ klickst.

Bitte beachten: Für die Installation der VPN Software von SpyOFF werden lediglich wenige MB benötigt.

Erzeuge eine Verknüpfung
66.666666666667% Complete

Im fünften Schritt des Installationsvorgangs erhältst Du die Möglichkeit Verknüpfungen zu erstellen. Indem Du ein Häkchen bei „Desktop“ setzt, kannst Du eine Verknüpfung auf Deinem Desktop erstellen. Mit einem Häkchen bei „Start Menu Programs folder“ erzeugst Du eine Verknüpfung im Startmenü. Bestätige bitte Deine Auswahl, indem Du auf „Next“ klickst und setze die Installation fort.

Bitte beachten: Das Erzeugen von Verknüpfungen ist nicht erforderlich, um die VPN Installation fortsetzen zu können, sondern lediglich optional und manuell konfigurierbar.

Starte die Installation
83.333333333333% Complete

Nachdem Du nun alle Vorbereitungen zur Installation getroffen hast, kannst du mit der Installation beginnen. Klicke hierzu auf den Button „Install“. Solltest Du nochmals Änderungen an der Installation vornehmen oder Dich vergewissern wollen, dass Du alles ordnungsgemäß eingestellt hast, klicke bitte auf „Back“.

Bitte beachten: Durch einen Klick auf „Cancel“ schließt Du den Wizard und brichst die VPN-Installation von SpyOFF ab.

Schließe die Installation ab
100% Complete

100%

Die VPN-Installation von SpyOFF ist nun abgeschlossen. Indem Du das Häkchen bei „Launch SpyOFF“ setzt, wird der VPN-Client sofort für Dich gestartet. Schließe bitte die Installation ab und klicke auf „Finish“.

Bitte beachte: Bei „Launch SpyOFF“ muss kein Häkchen gesetzt werden, um die VPN Installation beenden zu können. Die Wahl dieser Einstellung ist optional, wird jedoch empfohlen, um direkt mit der Nutzung von SpyOFF fortfahren zu können.

Großartig! Du hast die Installation von SpyOFF erfolgreich durchgeführt. Du kannst nun SpyOFF starten und eine VPN-Verbindung aufbauen!
